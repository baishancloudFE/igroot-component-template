# iGroot 示例组件

## 安装

### 1. 下载 npm 包

```js
npm i --save igroot-button
```

### 2. 在 bsy.json 中登记
```js
{
  "options": {
    "esModule": [
      "igroot-button"
    ]
  }
}
```

## 在代码中使用

参见 组件面板下的 JSX 面板

## API

| 参数        | 是否必填   | 说明    |  类型  |  默认值
| --------   | ----------:| -----:   | :----: |  :----: |
| type   | 非必填     | 按钮类型      |   string    | default
| title    | 非必填|   按钮上显示的文本    |   string    | xxx
